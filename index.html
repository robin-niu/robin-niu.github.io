Hexo发布错误：Deployer not found:github

2016-03-24
Hexo

Hexo发布错误：Deployer not found:github

使用HEXO向github更新文章，执行:

hexo deploy

出现以上提示，
需要将原来_config.yml配置中的deploy下的“type”字段中的“github”改为“git”。
再执行：

npm install hexo-deployer-git --save

重新执行hexo deploy即可。

官方设置可以参照：
https://github.com/hexojs/hexo-deployer-git
